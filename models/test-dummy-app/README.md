# Test Dummy App — Smoke-Test Model

A deliberately trivial model — a single `widget` concept with one seeded record — used to prove that a **second** application can be generated, deployed to its **own** Supabase instance and served on a separate port offset (`UBK_DEV_MODEL`). It exists only for the multi-app tests, not as a functional example; for that, see [test-app](../test-app/README.md).
