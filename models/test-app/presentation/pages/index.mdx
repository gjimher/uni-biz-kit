Welcome to the management portal.

## Public pages

Accessible to anyone, including unauthenticated users. Data is fetched via the Supabase `anon` role.

- [Model concepts](/example-mdx) — MDX page listing all model concepts
- [Product list — hooks](/example-data) — plain JSX using `useGetList`
- [Product list — RA components](/example-admin) — JSX with a React-Admin `Datagrid`

## Authenticated pages

These pages require login. The content is the same, but access is enforced via the `authenticated` Supabase role and the router's session guard, which bounces signed-out visitors to `/signin`.

- [Private MDX](/priv/example-private-mdx) — same concept list, requires auth
- [Product list — hooks (auth)](/priv/example-data) — same data, auth-only
- [Product list — RA components (auth)](/priv/example-admin) — same Datagrid, auth-only
